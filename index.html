aleph audio visualization for the masses

learn more

aleph is a framework for creating and performing audio-reactive visuals

approachable

aleph is built with modern web technologies and takes care of a lot of the heavy lifting, making it accessible to anyone who knows (or wants to learn) a bit of javascript.

extensible

100% open source and completely hackable. git your phreak on.

cross-platform

available for mac, windows, and linux. for free. aw yeah.

create

use aleph's powerful but approachable toolset to create beautiful visualizations. the sky is the limit.

perform your creations live with flexible audio anaylsis, midi, and multi-display support.
